Stock Analyzer Project - Complete File Listing

## Utils Directory (utilities)
```
utils/
├── indicators.py          # Technical indicators (RSI, MACD, Bollinger, SMA)
├── fundamentals.py        # Fetch fundamentals (P/E, Beta, Market Cap)
├── valuation.py           # Multi-method valuation (P/E, DCF, Gordon, P/B)
├── forecast.py            # Original forecast module (backup)
├── forecast_v2.py         # Advanced forecast with backtesting & sentiment
├── signals.py             # Trading signal generation (BUY/SELL/HOLD)
├── news.py                # News fetching & sentiment analysis (VADER)
└── models/               # (Created at runtime) Saved ML models

tests/
├── test_forecast.py       # Unit tests for forecast module
```

## Key Python Files

### app.py (Main Application)
- Streamlit UI framework
- Data fetching (yfinance)
- Chart rendering (Plotly)
- Sidebar controls
- Metrics display
- ~240 lines

### utils/indicators.py
- RSI (14-period)
- MACD with signal line
- Bollinger Bands
- Simple Moving Averages (50, 200)
- Fallback manual RSI computation
- ~60 lines

### utils/fundamentals.py
- Fetch P/E (Trailing & Forward)
- Beta coefficient
- Dividend Yield
- Market Cap
- Using yfinance API
- ~20 lines

### utils/valuation.py (NEW)
- P/E Multiple valuation
- Gordon Growth Model
- DCF (Discounted Cash Flow) with 5-year forecast
- Price-to-Book method
- Multi-method consensus
- ~140 lines

### utils/forecast_v2.py (NEW)
- RandomForest model
- ARIMA placeholder
- LSTM placeholder
- Lag feature creation
- Sentiment feature integration
- Walk-forward backtesting
- RMSE, MAE, Direction Accuracy metrics
- ~130 lines

### utils/signals.py
- RSI-based signal logic
- SMA crossover logic
- BUY/SELL/HOLD classification
- ~45 lines

### utils/news.py (NEW)
- NewsAPI integration
- VADER sentiment analysis
- Daily sentiment aggregation
- Offline demo mode
- Sentiment feature creation
- ~90 lines
